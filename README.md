# StayAI - Customer Retention Intelligence

This repository contains the StayAI frontend and the shareable Python source files used for customer retention analytics workflows.

## Included

- `frontend/` - Vite + React dashboard UI
- `stayai-ml-service/` - Python scripts for data generation, training, and SHAP enrichment

## Excluded

The repository intentionally does not track local environments, generated build output, datasets, model binaries, exported presentations, or other large/sensitive artifacts.
